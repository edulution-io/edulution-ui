Look up actual TypeScript types from the codebase before writing factories:

- `userFactory.ts`: Look for the User/UserDto type in `libs/src/user/` or `libs/src/auth/`. Use German school names (e.g., "Realschule Musterstadt"), LDAP groups (e.g., "cn=teachers,ou=groups,dc=schule,dc=de"). Also export `createJwtUser` for JWT token payloads.
